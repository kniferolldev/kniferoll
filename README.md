# kniferoll

To install dependencies:

```bash
bun install
```

To run the CLI:

```bash
bun run index.ts
```

To launch the browser demo:

```bash
bun run demo
```

Then open one of:

- `http://127.0.0.1:5173/minimal` for the barebones embed (`<script>` + `<kr-recipe>` inline markdown).
- `http://127.0.0.1:5173/` for an interactive playground with recipe/scale/quantity selectors.

This project was created using `bun init` in bun v1.2.21. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
